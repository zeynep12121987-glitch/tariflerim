Client ID Nasıl Alınır?

Bu uygulama Drive'ına erişmek için Google'ın ücretsiz API'sini kullanır. Bir kez kurulur, bir daha dokunmazsın.

1
Tarayıcında console.cloud.google.com aç
2
Yeni Proje oluştur → istediğin ismi ver
3
Sol menü → API ve Hizmetler → Kitaplık → "Google Drive API" ara → Etkinleştir
4
Kimlik Bilgileri → Kimlik Bilgisi Oluştur → OAuth 2.0 İstemci Kimliği
5
Uygulama türü: Web Uygulaması → Yetkili JavaScript kaynağı: bu sayfanın URL'ini ekle
6
Oluşturulan Client ID'yi kopyala → uygulamaya yapıştır
